# Extensible Payment System

A Java-based demo project that showcases how to integrate a new payment method into an existing payment flow while following SOLID principles, especially OCP(Open-Closed Principle) and SRP(Single-Responsibility Principle).

## Goals

- Keep the existing system as stable as possible
- Add new payment methods in an extensible way
- Use Chain of Responsibility(CoR) for payment flow orchestration
- Use Reflection for dynamic payment processor discovery
- Provide a clean and understandable architecture

## Tech Stack

- Java 21
- Maven

## Status

Project skeleton is being built incrementally with small, meaningful commits.
